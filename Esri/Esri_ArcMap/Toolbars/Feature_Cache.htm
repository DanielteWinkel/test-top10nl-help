Feature Cache

Als je met data werkt die is opgeslagen in een geodatabase kan het maken van de map cache de snelheid van ArcMap taken verhogen. Met de map cache kun je tijdelijk objecten in de actieve extent in het interne geheugen van je computer opslaan. Omdat het raadplegen van objecten in het lokale geheugen heel snel gaat, kan het gebruik van de feature cache de prestaties vaak verbeteren.

 

De feature cache slaat alleen objecten uit een geodatabase op, dus geen data van rasters. De feature cache is het meest effectief als je binnen een bepaald gebied gaat werken. Bijvoorbeeld, als je werkt met data over een netwerk die opgeslagen is in een multiuser database. Objecten in de actieve extent moeten opgehaald worden uit die database elke keer dat je het beeld ververst. Het gebruik van de feature cache kan de belasting van het netwerk verminderen, omdat ArcMap de objecten uit het interne geheugen (RAM) van je computer haalt. Omdat objecten opgeslagen zijn aan de cliënt kant, vermindert het aantal bevragingen van de server. Het tekenen van complexe datasets, labeling, bewerken, selecteren en het ophalen van dezelfde objecten voor meerdere lagen in de kaart zijn activiteiten die kunnen profiteren van de feature cache. Labeling bijvoorbeeld kan soms een traag proces zijn voor de database, omdat het nodig is dat de database meerdere keren geraadpleegd wordt voor het plaatsen van vele labels in de kaart.

 

 

Het gebruik van de feature cache tools

ArcMap heeft tools om je te helpen met het opbouwen van de feature cache. Deze tools kun je vinden in de Feature Cache toolbar. Je kunt een feature cache aanmaken door op de knop Build Map Cache te klikken . Je kunt ook de automatische cache gebruiken om de cache op te bouwen zodra je het beeld in de extent verandert. De automatische cache kan nuttig zijn als je in verschillende gebieden werkt en je niet elke keer handmatig een cache wilt opbouwen. Omdat het automatisch opbouwen van de cache de prestaties kan hinderen is het verstandig om een minimum schaal in te stellen. Als je bijvoorbeeld de minimum schaal instelt op 1:5000 en je zoomt in tot 1:4500 zal de feature cache automatisch worden aangemaakt. Als je uitzoomt tot 1:7500 zal de cache niet gemaakt worden.

 

Maximaliseer de prestaties van de feature cache

Als je met een enterprise database werkt, zoals we dat bij het Kadaster doen, kan het consequent gebruik maken van de feature cache de prestaties enorm verbeteren doordat het aantal raadplegingen van de database afneemt en de daarmee ook het netwerkverkeer.

Het gebruik van de feature cache op een personal database heeft minder voordelen. Wel als je objecten met heel veel punten zichtbaar in de extent zichtbaar hebt en snapping gebruikt. Als je alleen maar veel zoomt in een groot gebied heb je niet zoveel voordelen van de feature cache...

 

Hoe werk je met de feature cache

Open de Feature Cache toolbar:

 

Een feature cache maken

  1. Pan of zoom naar het gebied waarin je wil werken

  2. Klik de knop Build Feature Cache . De objecten in de extent worden in het interne geheugen van je computer geplaatst.

 

  • Omdat de feature cache is opgeslagen in het interne geheugen van je computer kan het maken van een cache met veel objecten veel geheugen vragen en kan het ook lang duren. Je kunt tijdens het maken van de cache op de ESC knop drukken om het proces te onderbreken.
  • Je kunt op ieder moment snel naar de feature cache extent terugkeren door de knop Zoom To Map Cache te klikken.
  • Je kunt op de knop Show Map Cache klikken om het gebied van de actieve cache zichtbaar te maken. Je maakt alleen gebruik van de feature cache als de knop Show Map Cache groen is. Als de knop rood wordt betekent dit dat je gedeeltelijk buiten het gebied bent en geen gebruik meer maakt van de cache. Als de knop uitgegrijsd is ben je geheel buiten het gebied.
  • Als een gedeelte van je huidige extent buiten de cache valt, wordt de cache niet meer gebruikt. Om de cache weer te gebruiken moet je hem opnieuw opbouwen of de automatische cache inschakelen.
  • Om de cache te legen klik je op de knop Empty Map Cache .

 

De minimum schaal instellen voor de automatische cache
  1. Zoom in of uit naar de schaal waarop je gaat werken
  2. Klik de knop Set Auto-Cache Scale .
  • Als je het afgebeelde gebied in de extent wijzigt en je bent nog steeds in het gebied dat in de cache zit, wordt de cache niet opnieuw opgebouwd. Als het afgebeelde gebied niet geheel in de cache zit, zal de cache weer automatisch worden opgebouwd, ten minste als je niet bent uitgezoomd buiten de minimum schaal.
  • Om te voorkomen dat je niet van de gehele database een cache maakt is het verstandig een minimum schaal in te stellen. De cache wordt dan niet automatisch opgebouwd.

 

Automatische cache opties instellen
  1. Ga naar de Data Frame Properties
  2. Selecteer het tabblad Map Cache
  3. Vink de optie Automatically create feature cache aan
  4. Vink de optie Set minimum scale for auto-cache aan
  5. Geef een minimum schaal op
  6. Klik OK

 

De minimum schaal voor de automatische cache wissen

Klik op de knop Clear Auto-Cache Scale .

 

Het gebied bekijken dat in de cache zit

  1. Klik de knop Show Map Cache. Het gebied dat in de cache zit wordt flitsend zichtbaar gemaakt in de extent.

    Als de knop Show Map Cache groen is gebruik je de feature cache.

    Als de knop Show Map Cache rood is is een deel van het gebied in de extent niet opgenomen in de cache.

    Als de knop niet beschikbaar is (uitgegrijsd) is het afgebeelde gebied in z’n geheel niet in de cache opgenomen

  2. Optioneel, als je van plan bent om toch in het afgebeelde gebied te gaan werken, maak dan opnieuw de cache aan op zet de automatische cache aan.

  • De knop Show Map Cache is alleen toegankelijk als je een feature cache heb aangemaakt en het afgebeelde gebied in de extent overlapt met het gebied in de cache.
  • Als een deel van het afgebeelde gebied in de extent buiten het gebied van de cache valt maak je geen gebruik meer van de cache. Om wel weer gebruik te maken van de cache moet je deze opnieuw opbouwen, de automatische cache inschakelen of terug gaan naar het gebied dat in de cache zit.
  • Je kunt snel naar het gebied van de cache gaan door de knop Zoom to Map Cache te klikken
