Default database instellen

Er moet gecontroleerd worden of de Top10NL-productie-database is ingesteld als “default database”, om te kunnen werken met de Workflow Manager.

1

Ga naar het Start menu, klik op All Programs en klik op ArcGIS -> ArcCatalog10.1

2

ArcCatalog wordt gestart. Ga in de Catalog Tree naar Workflow Manager Databases -> TTNL_WP Klik rechts en controleer of er vinkje staat voor Set as default database.

3

Mocht dit niet het geval zijn, dan moet tijdelijk de Workflow Manager extensie aangezet worden om de Set-as-default-database-optie te enable-n. Zoals je ziet is deze nu disabled (uitgegrijsd).

Ga in de menubar naar Customize -> Extensions…

4

Open Arcatalog en onder Customize en dan extensions, zet een vinkje bij de workflow manager en sluit het venster weer (Close).

5

Ga opnieuw in de Catalog Tree naar Workflow Manager Databases -> TTNL_WP Klik rechts, en klik op Set as default database.

Belangrijk:

Vink nu, via het extensions-dialoogvenster, Workflow Manager weer uit !

Het aanzetten wordt vanaf ArcGIS 10.1 namelijk vanuit de WorkflowManager-applicatie zelf geregeld. Als de gebruiker deze zelf in ArcCatalog aanzet, kan er een licensieprobleem ontstaan.

Snaptolerantie

De snaptolerantie is de afstand waarbinnen de muisaanwijzer springt naar een punt. Als het punt waarnaar gesprongen wordt (tussenpunt, rand, of eindpunt) binnen de afstand is die je hebt ingesteld, springt de muisaanwijzer automatisch naar dat punt (snapping).

De cirkel rond de muisaanwijzer geeft aan als er gesprongen wordt.

Als het punt waarnaar gesprongen dient te worden buiten de tolerantie valt zal de cirkel bij de aanwijzer blijven: .

Als het punt binnen de tolerantie komt verplaatst de cirkel zich naar het punt:.

Als je tijdens het tekenen de T-toets indrukt wordt d.m.v. een cirkel de snaptolerantie

zichtbaar.

 

Snaptolerantie instellen via dialoogvenster

  1. Open de toolbar Snapping: .

  2. Klik op snapping en daarna op options om een snapping waarde te wijzigen: .

 

Snaptolerantie interactief instellen
  1. Klik de knop Snap Tolerance .
  2. Geef een datapunt in de kaart en houdt de muisknop ingedrukt en beweeg de muis om een cirkel te tekenen.

Als je wel een datapunt geeft, maar geen cirkel tekent wordt de snaptolerantie ingesteld op 0.

Een snaptolerantie van 0 kan de prestaties dramatisch verminderen.

 

Sticky move tolerantie

Sticky move tolerance is afstand die de muisaanwijzer moet bewegen alvorens een geselecteerd object wordt verplaatst. Bij een ingestelde waarde van 0 loop je het risico dat een object al verplaatst tijdens het selecteren. Als je een waarde van 15 of meer instelt is het risico aanzienlijk verminderd. Als je een te grote waarde ingeeft kun je geen object meer verplaatsen.

 

Sticky move tolerance instellen via dialoogvenster

1

Open het Editing Options venster door op de knop Options . te klikken

2

Selecteer het tabblad General.

3

Pas je Sticky move tolerance aan (meeste topografen houden 75 of 100 pixels aan).

Haal het vinkje weg bij Use symbolyzed feature during editing.

Dit zorgt ervoor dat je tijdens het editen van een vlak, je luchtfoto blijft zien onder dat vlak.

Zet vinkje bij Use classic snapping

4

Klik OK.

Optimale waarde voor de toughbook in de buitendienst is 75

 

Snappen naar kaartlagen (layers) en Edit Sketch

Je kunt aangeven naar welke kaartlagen (layers) je wil snappen. Je kunt niet aangeven naar welke afzonderlijke objecten je wil snappen. Per kaartlaag kun je instellen naar welk deel van de objecten je wil snappen, tussenpunten (vertices), randen (edges) of endpoints (eindpunten). De snapinstellingen per kaartlaag noem je "snapping properties". Ook kun je instellen of je wil snappen naar de sketch, tussenpunten en randen en of je haaks (penpendicular) op de sketch wil kunnen tekenen.

 

Snapping properties instellen

 

Precondities:

Een edit sessie moet zijn gestart

 

  1. Open het Snapping Environment venster met de knop
  2. Stel voor elke kaartlaag de snapping in.

  3. Stel de snapping in voor de Edit Sketch.

    De instellingen worden meteen actief.

  • Je kunt de snapvolgorde van de lagen instellen door in het Snapping Environment venster de lagen met de muis te verschuiven.

  • Naast het Snapping Environment wordt ook het venster van de Snappig Wizard geopend, waarin je voorkeuren kunt opslaan om zo snel veel instellingen te kunnendoen.

Selectie tolerantie

De selectie tolerantie is de afstand in pixels t.o.v. de muisaanwijzer waarbinnen objecten kunnen worden geselecteerd. Standaard staat een waarde van 3 pixels ingesteld. Als je soms moeite hebt een lijnobject te selecteren kan een hogere waarde uitkomst bieden

 

Selection tolerance instellen

  1. Open het Selection Options venster door met de knop .
  2. Stel een waarde in voor Selection tolerance
  3. Klik OK

Feature cache

Het gebruik van de map cache kan het werken aanzienlijk versnellen.

Voor het werken met een map cache is een aparte toolbar aanwezig.

De map cache kun je handmatig of automatisch aanmaken.

 

Zie Feature Cache voor een uitgebreide beschrijving.

 

Handmatig een map cache maken
  1. Open de Feature Cache toolbar
  2. Pan of zoom naar het gebied waarin je wil werken
  3. Klik de knop Build Map Cache . De objecten in je extent worden in het interne geheugen van je computer geplaatst.

 

Automatisch een map cache aanmaken
  1. Open de Map Cache toolbar.
  2. Pan of zoom naar het gebied waarin je wil werken.
  3. Klik de knop Toggles Auto-Cache . De objecten in de extent worden in het interne geheugen van je computer geplaatst telkens als het kaartbeeld in de extent veranderd.

Vergeet niet een Auto-Cache schaal in te stellen.

 

Schaal instellen voor de automatische map cache
  1. Zoom in of uit naar de schaal waarop je gaat werken.
  2. Klik de knop Set Auto-Cache Scale

 

De minimum schaal voor de automatische cache wissen

Klik op de knop Clear Auto-Cache Scale

 

De map cache leeg maken
  1. Klik op de knop Empty Map Cache

Zet alle objecten in de cache door eerst naar de AOI te zoomen en daarna de cache op te bouwen

 

Map topology

Map topology is een simpele topology die je kunt toepassen op simpele objecten (punten, lijnen en vlakken) tijdens een edit sessie. Met map topology kun je tegelijkertijd meerdere objecten bewerken die elkaar overlappen of raken.

Je kunt de Topology Edit tool , de Modify Edge en Reshape Edge taken gebruiken om objecten te bewerken. De objecten kunnen zich in verschillende objectklassen (featureclasses) bevinden en verschillende geometriën hebben. Lijnobjecten en de randen van vlakobjecten worden topologische segmenten (edges) als je map topology gebruikt. Puntobjecten, de eindpunten van lijnen en de plaatsen waar segmenten elkaar snijden worden knooppunten (nodes).

 

Zie Map topologie voor een uitgebreide beschrijving.

 

Map topology instellen:

 

Precondities:

Een edit sessie moet zijn gestart

 

  1. Klik de Map topology knop in de Topology toolbar.

  2. Vink de objectklassen aan die mee moeten doen in de map topology.

  3. Alle objectklassen in de actieve edit sessie, behalve annotatie en maatvoering, kunnen meedoen en zijn zichtbaar in het dialoogvenster.

  4. Optioneel, stel een cluster tolerantie in van 0.15 mtr.

  5. De standaard cluster tolerantie is de kleinst mogelijke cluster tolerance. Het vergroten van de cluster tolerantie kan leiden tot het samenvoegen van meer objecten die als samenvallend worden beschouwd, maar kan onnauwkeurigheden in de hand werken.

  6. Klik OK.
