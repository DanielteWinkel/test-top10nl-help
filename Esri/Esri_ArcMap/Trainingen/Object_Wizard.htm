Object Wizard

Met de ObjectWizard is het mogelijk om snel objecten te digitaliseren. Na selectie van een gewenst object worden er allerlei instellingen gezet in de editor omgeving om het mogelijk te maken een nieuw object te digitaliseren. Tevens wordt ervoor gezorgd dat het nieuwe object (feature) naast de gedigitaliseerde geometrie een attribuut set krijgt met waarden overeenkomstig de gekozen object definitie. De ObjectWizard maakt het ook mogelijk om bestaande objecten de wijzigen. De object wizard bevat een keuze lijst van objecten. Uit deze lijst kan de gebruiker een object kiezen en dan een nieuw object in de kaart schetsen. Wanneer een object uit de lijst gekozen is, zorgt de wizard ervoor dat de goede editor instellingen in ArcMap worden ingesteld. Hierdoor kan de gebruiker de schets intekenen, ArcMap zorgt ervoor dat deze in de goede laag in de database bewaard wordt en dat tevens de standaard voorgedefinieerde attributen worden ingevuld.

 

De ObjectWizard maakt het tevens mogelijk om met EspaCity als meetsysteem te werken. De dialoog van de ObjectWizard kan op het Espa scherm geplaatst worden waardoor het lijkt of het bij EspaCity hoort. Geometrieën die vervolgens in EspaCity gedigitaliseerd worden, worden onder aansturing van de ObjectWizard in ArcMap omgezet in het gekozen objecttype (feature). De nauwe koppeling met Espa City zorgt er ook voor dat instellingen en digitaliseer commando’s naar EspaCity verstuurd kunnen worden.

 

De ObjectWizard bestaat uit een set object definities die zijn gegroepeerd in categorieën.

 

Dialoog

De ObjectWizard dialoog toont alle categorieën met daarin de bijbehorende objectdefinities. Uit deze lijst kan een definitie gekozen worden die vanaf dat moment als actueel object aangemerkt wordt. Elk nieuw gemaakte feature krijgt vanaf dat moment de eigenschappen (attributen) van dat actuele object. De dialoog is opgenomen in de Table of Contents (TOC) scherm van ArcMap.

 

Werking

De objectWizard bestaat uit een actief en een passief deel. In het actieve deel worden er een aantal acties uitgevoerd. Direct na selectie van een definitie in de wizard worden de bijbehorende acties in deze volgorde uitgevoerd :

  • selecteer de target layer
  • selecteer de Edit Task
  • selecteer en activeer de Sketch Tool
  • verstuur EspaCity commando’s

 

De ObjectWizard wordt pas actief als “Start Editing” gebruikt wordt. En wordt weer inactief bij “Stop Editing”.

 

Knoppen en knoppenbalken

 

Elke object definitie in de lijst in de ObjectWizard is ook als knop beschikbaar. Deze knoppen kunnen via ArcMap customize op knoppenbalken geplaatst worden.

Aan elke object definitie kan in de configuratie een eigen icoon/bitmap gekoppeld worden.

 

Top10NL Top10

De ObjectWizard heeft een “Top 10” knoppenbalk met daarin de laatste 10 , door de gebruiker, gebruikte ObjectDefinitie knoppen. Als er een knop geactiveerd wordt, wordt gekeken of deze al in de Top 10 balk voorkomt, zoniet dan wordt deze aan de rechterkant toegevoegd. Als er meer dat 10 knoppen in de balk staan dan valt bij het toevoegen aan de rechter kant de eerste knop aan de linkerkant uit de balk. Bij het starten is de balk leeg. De inhoud wordt niet tussen de sessies opgeslagen

Sketchtool button
