Een lijnobject construeren

Lijnobjecten kun je construeren door de geometrie van een bestaand(e) object(en):

  • te volgen met de standaard Trace Tool .
  • parallel te kopiëren met de standaard Copy Parallel Tool (zie Top10NL Editing Advanced).
  • op te delen met de Cut Geometry taak of met de standaard Split Tool.
  • op te delen en meteen attributen wijzigen met de Top10NL Cut Object taak
  • samen te voegen tot een nieuw object met de Mingle tool of de Top10NL tool Create Object (zie Top10NL Editing).

 

Een lijnobject tekenen

Met de sketch tool wordt een schets gemaakt (edit sketch). Een schets bestaat uit punten (vertices) en lijnen (segments) die deze punten verbinden.

Het actieve objecttype (featureclass) bepaalt welk type object getekend wordt.

Pas na het accepteren van de schets (Finish sketch) wordt een object in de database gemaakt.

1

Stel met de Object Wizard het actuele objecttype in.

In dit geval het type lijn.

De sketch tool wordt automatisch gestart.

2

Teken de gewenste punten van de lijn.

Klik Finish Sketch om het lijnobject op te slaan.

3

Het lijnobject is toegevoegd en nog geselecteerd.

4

Klik eventueel op Clear selected Features om de selectie op te heffen.
  • Om de schets te bevestigen (Finish sketch ) kun je ook F2 gebruiken of dubbel klikken met de linker muisknop.

  • Hef een selectie niet te snel op.

    Vaak wil je meteen met het getekende object nog een actie uitvoeren zoals invoegen (zie Geometry Wizard),

    of wil je nog attributen wijzigen met de Attribute Editor.

 

Een puntobject tekenen

Met de sketch tool wordt een schets gemaakt (edit sketch). Een schets bestaat uit punten (vertices) en lijnen (segments) die deze punten verbinden.

Het actieve objecttype (featureclass) bepaalt welk type object getekend wordt.

Om een punt te tekenen is 1 klik voldoende. Om een lijn of vlak te tekenen moet je de schets gebruiken.

Pas na het accepteren van de schets (Finish sketch) wordt een object in de database gemaakt.

1

Stel met de Object Wizard het actuele objecttype in.

In dit geval het type punt.

De sketch tool wordt automatisch gestart.

2

Een enkel datapunt is voldoende om een puntobject te plaatsen.

Het punt blijft geselecteerd.

3

Klik eventueel op Clear selected Features om de selectie op te heffen.

  • Hef een selectie niet te snel op.

    Vaak wil je meteen met het getekende object nog een actie uitvoeren zoals invoegen (zie Geometry Wizard),

    of wil je nog attributen wijzigen met de Attribute Editor.

 

Een vlakobject construeren

 

Construeren zonder objecten te selecteren

Zonder een object te selecteren kun je een lege ruimte opvullen met de Top10NL Auto-Complete Polygon taak .

 

Construeren m.b.v. geselecteerde objecten

Selecteer met de Edit Tool of 1 van de Select tools (zie Top10NL Select) altijd eerst 1 of meerdere objecten die je wilt gebruiken.

 

Na het selecteren kun je objecten gaan construeren door de geometrie van een geselecteerd(e) object(en):

  • te gebruiken met de Buffer tool .
  • over te nemen met de Top10NL tool Object Creator (zie Top10NL Editing).
  • op te delen met de Top10NL Cut Geometry taak .
  • op te delen en meteen objecttype of attributen wijzigen met de Top10NL Cut Object taak .
  • gedeeltelijk te gebruiken voor een nieuw object met de Top10L Autocomplete selection polygon taak .
  • samen te voegen tot een nieuw object met de Mingle tool of de Top10NL tool Create Object (zie Top10NL Editing).

 

Een vlakobject tekenen

Met de sketch tool wordt een schets gemaakt (edit sketch). Een schets bestaat uit punten (vertices) en lijnen (segments) die deze punten verbinden.

Het actieve objecttype (featureclass) bepaalt welk type object getekend wordt.

Pas na het accepteren van de schets (Finish sketch ) wordt een object in de database gemaakt.

1

De sketch tool wordt automatisch gestart.

2

Teken de gewenste punten van het vlak.

Klik op Finish Sketch om het vlakobject op te slaan

3

Het vlak is toegevoegd en is geselecteerd.

4

Klik eventueel op Clear selected Features om de selectie op te heffen.
  • Om de schets te bevestigen (Finish sketch ) kun je ook F2 gebruiken of dubbel klikken met de linker muisknop.

  • Hef een selectie niet te snel op.

    Vaak wil je meteen met het getekende object nog een actie uitvoeren zoals invoegen (zie Geometry Wizard),

    of wil je nog attributen wijzigen met de Attribute Editor.

  • Kijk hier hoe je een Vlakobject met gat tekent .

 

Een vlak maken met de buffer tool

1

Selecteer een bestaande lijn of breng een nieuwe lijn aan.

2

Selecteer in de Object Wizard de objectdefinitie van het te maken vlak.

3

Bedenk nu al hoe breed het te maken vlak moet worden.

Selecteer de Buffer Tool uit de Top10NL Editing Advanced toolbar (de buffertool uit de Editor mag niet gebruikt worden; deze veroorzaakt fouten) en geef de gewenste breedte op in het venster dat verschijnt.

 

Vergeet niet een ENTER te geven zodat het venster sluit en de buffer wordt toegepast.

4

Het vlak is gemaakt en is geselecteerd.

5

Klik eventueel op Clear selected Features om de selectie op te heffen.

 

Een gat aanbrengen

Enkele mogelijkheden om een gat aan te brengen in een bestaand vlakobject:

 

m.b.v. de Geometry Wizard

Afhankelijk van het in te voegen object kun je met de Geometry Wizard een gaat maken in het onderliggende vlakobject.

Hieronder een voorbeeld van een watervlak dat ingevoegd wordt in een terrein.

1

Stel met de Object Wizard het actuele objecttype in.

De sketch tool wordt automatisch gestart.

2

Teken de gewenste punten van het vlak.

Klik op Finish Sketch om het vlakobject op te slaan.

3

Het vlak blijft geselecteerd.

Start de tool Geometry Wizard (stanser) om het onderliggende terrein uit te snijden.

4

Verwijder indien nodig het watervlak met Delete .

 

m.b.v. de taak Cut Polygon Features

1

Selecteer het te bewerken vlakobject met de Edit tool .

2

Selecteer de taak Cut Polygon Features in de Editor toolbar.

Selecteer de sketch tool .

3

Teken de vorm van het gat.

Zorg ervoor dat het begin- en eindpunt van de sketch samenvallen.

Maak de sketch af met Finish sketch .

4

Selecteer nu het object dat in het gat zit met de Edit tool en verwijder het object met Delete .

5

Het resultaat.

 

m.b.v. de Clip tool

Het gebruik van de tool wordt afgeraden.

Deze tool verwijdert namelijk ook delen van objecten in de lagen Functioneel gebied en Geografisch gebied.

Gebruik de Geometry Wizard of de Cut Geometry taak.

 

door punten te verleggen met de taak Modify Feature

1

Het te wijzigen vlakobject.

2

Dubbelklik het te bewerken vlakobject met de Edit Tool om de vertices (buigpunten) zichtbaar te maken

of

selecteer de taak Modify Feature.

3

Verplaats de punten als aangegeven.

4

Het resultaat.

 

Een gat verwijderen

Er zijn 3 mogelijkheden om een gat te verwijderen, met behulp van:

 

Edit tool

1

Klik de Edit Tool en dubbelklik het vlakobject dat je wilt gebruiken.

De taak Modify Feature is gestart.

2

Verwijder 1 voor 1 de punten van het vlak door met de

rechtermuisknop te klikken op de punten en Delete Vertex te kiezen.

3

Als alle punten zijn verwijderd klik je op Finish Sketch .

4

Er is 1 geometrie overgebleven.

 

Edit Sketch Properties

1

Klik de Edit Tool en dubbelklik het vlakobject dat je wilt gebruiken.

De taak Modify Feature is nu ingesteld.

2

Open het Sketch Properties venster en selecteer alle vertices van het gat en klik dan op Delete .

Klik Finish Sketch om de gewijzigde geometrie te bevestigen.

3

Er is 1 geometrie overgebleven.

 

Een extra object en de Mingle tool

1

Stel met de Object Wizard het actuele objecttype in.

Kies hetzelfde type als het object met het gat erin.

De sketch tool wordt automatisch gestart.

2

Teken een vlak dat het gat ruim bedekt.

Klik op Finish Sketch om het vlakobject op te slaan.

3

Start de Edit tool en selecteer zowel het object met het gat als het zojuist getekende object.

4

Selecteer de Mingle tool en selecteer in het merge dialoogvenster het object waarvan de attributen worden overgenomen en klik OK.

5

Beide geometriën zijn samengevoegd en er is 1 object overgebleven.

Geometrie wijzigen

De geometrie van een vlak- of lijnobject wijzig je doorgaans met:

 

De Edit Tool

1

Selecteer de Edit Tool en dubbelklik het object om de tussenpunten zichtbaar te maken.

De taak Reshape Feature is gestart.

2

Verplaats de tussenpunten zoals gewenst.

Voeg eventueel extra punten toe door met de aanwijzer op de juiste plaats te gaan staan en de rechter muisknop te klikken.

3

Hef eventueel de selectie op .

De geometrie is aangepast.

 

DeTopogy Edit Tool

 

Preconditie

Selecteer de lagen die mee moeten doen in de topologie . Zie Map topologie.

 

1

Selecteer de Topology Edit Tool en dubbelklik de geometrie (rand) van het object om de tussenpunten zichtbaar te maken.

De taak Reshape Feature is actief geworden.

2

Verplaats de tussenpunten zoals gewenst.

Voeg eventueel extra punten toe door met de aanwijzer op de juiste plaats te gaan staan en de rechter muisknop te klikken.

3

Deselecteer de geometrie .

De geometrie is aangepast.

Let er wel op dat je van nieuwe objecten de attributen weer correct invult.

Objectklasse wijzigen

Als je een terreinvlak wijzigt in een watervlak wijzigt de Objectklasse van het object.

 

Een terreinvlak wijzigen in watervlak met de ObjectWizard

1

Selecteer het te wijzigen vlakobject met de Edit tool .

Stel met de Object Wizard het actuele objecttype in. In dit geval het type vlak.

2

Klik de knop Object wijzigen om het geselecteerde vlak te wijzigen.

3

Het resultaat nadat de selectie is opgeheven met Clear selected Features.

Je kunt met de ObjectWizard alleen wijzigen als het geometrytype (lijn, vlak, punt) van het geselecteerde object gelijk is aan die van de ingestelde objectdefinitie.

M.a.w. je kunt geen watervlak wijzigen in een weglijn en omgekeerd.

 

Een terreinvlak wijzigen in een waterlijn met Copy en Paste

1

Selecteer het te wijzigen vlakobject met de Edit tool .

Stel met de Object Wizard het actuele objecttype in. In dit geval het type waterlijn.

Verplaats het terreinvlak naar het interne geheugen met de Cut tool (Main Menu->Edit->Cut)

2

Plak het gekopieerde vlakobject met Paste .

ArcMap zorgt ervoor dat de geometrie van het vlak wordt gebruikt om een lijn te maken

3

Het resultaat nadat de selectie is opgeheven met Clear selected Features.

Je kunt op deze manier niet een vlak maken van een gekopieerde lijn.
