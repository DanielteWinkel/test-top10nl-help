Landart

Definitie

Grootschalige kunstzinnige landschapsprojecten.

Zie ook Markant object

 

Regelgeving

Grootschalige, permanent aanwezige, landart-objecten met eigennaam die niet voldoen aan de maatvoering voor markant object kunnen worden opgenomen als inrichtingspunt van het type overig.

 

De eigennaam dient als attribuutwaarde naam te worden toegevoegd. Wanneer er geen naam bekend is wordt het object niet ingewonnen.
