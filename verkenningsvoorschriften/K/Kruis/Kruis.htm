Kruis

Definitie

Object met of in de vorm van een kruis, dat dient als uiting van religieuze toewijding, ter bezinning, herdenking en/of verering.

 

Regelgeving Topografie

De topografie wordt aangegeven volgens fotobeeld.

 

Regelgeving Thematisch Proces

Dit type wordt bijgehouden in een apart thematisch proces, zie Externe bronnen / data.

 

Criteria

  • Een object met of in de vorm van een kruis geven we aan als inrichtingselement van het type kruis.
  • Een object in een andere vorm dan een kruis geven we aan als inrichtingselement van het type overig met soortnaam religieus object. Zie Religieus object.

  • Een gedenkteken met of in de vorm van een kruis, geven we aan als ‘gedenkteken, monument’. Zie Gedenkteken.

 

Externe bron

 

Naam

Een eigennaam kan als attribuutwaarde naam aan het betreffende object worden toegevoegd.
