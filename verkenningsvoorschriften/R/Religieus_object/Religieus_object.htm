Religieus object

Definitie

Object in een andere vorm dan een kruis, dat dient als uiting van religieuze toewijding, ter bezinning, herdenking en/of verering.

 

Zie ook Bebouwing met religieuze bestemming, Kapel, bidkapel, Kruis

 

Regelgeving Topografie

De topografie wordt aangegeven volgens fotobeeld.

 

Regelgeving Thematisch Proces

Dit type wordt bijgehouden in een apart thematisch proces, zie Externe bronnen / data.

 

Criteria

  • Een nis of kastje langs de weg, losstaand of aan de muur van een gebouw, met een Maria- of heiligenbeeld, geven we aan als inrichtingselement van het type overig met soortnaam religieus object

  • Een standbeeld van een heilige geven we aan als inrichtingselement van het type overig met soortnaam religieus object.

  • Een kruisweg geven we aan als inrichtingselement van het type overig met soortnaam religieus object
  • Een grot of grotachtig object met een Maria- of heiligenbeeld geven we aan als inrichtingselement van het type overig met soortnaam religieus object
  • Een object met of in de vorm van een kruis geven we aan als inrichtingselement van het type kruis. Zie Kruis.
  • Kleinere attributen of versieringen aan gevels nemen we niet op.
